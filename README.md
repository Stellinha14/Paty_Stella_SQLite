# App utilizando SQLite :iphone:

**Aplicativo em React com SQLite desenvolvido no 3° do curso técnico em Informática para Internet**

---

## Instalação das Dependências

Para instalar todas as dependências do projeto, utilize os comandos abaixo:

- [ ] `npm install`
- [ ] `npx expo install expo-sqlite`

---

## Manipulação de Imagens

O projeto utiliza manipulação de imagens, portanto é necessário instalar a biblioteca adequada.

**Comando para instalar a biblioteca de imagens:**

- [ ] `npx expo install expo-image-picker`

---

## Inicialização do Projeto

Para iniciar o aplicativo, utilize:

`npx expo start`
